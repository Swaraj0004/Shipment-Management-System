# SwiftBL MERN Assignment

Shipment Management System built using MERN stack.

## Features
- Create shipment
- View shipments
- Update shipment status
- Delete shipment

## Tech Stack
MongoDB, Express.js, React.js, Node.js

## Setup

### Backend
cd backend  
npm install  
npm run dev  

### Frontend
cd frontend  
npm install  
npm run dev
