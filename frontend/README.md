# SwiftBL MERN Assignment

A simple Shipment Management System built using the MERN stack as part of the SwiftBL internship assignment.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- Create shipment
- View all shipments
- Update shipment status
- Delete shipment

## Project Structure
```
swiftbl-mern-assignment/
├── backend/
│   ├── models/
│   ├── routes/
│   ├── server.js
│   └── .env.example
├── frontend/
│   └── src/
└── README.md
```

## Setup Instructions

### Backend
```bash
cd backend
npm install
npm run dev
```

Create a `.env` file using `.env.example`.

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Notes
This project focuses on implementing a clean and working MERN architecture within a time-bound assignment window.  
The structure and APIs are designed to be easily extendable with authentication, validations, and additional business logic.
